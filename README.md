# Sales Performance Analysis using Excel

## Objective
To analyze sales data and identify top-performing regions, products, and salespersons.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas

## Dataset
Sales dataset containing order date, region, product, salesperson, quantity, and price.

## Analysis & Insights

### Region-wise Sales
![Region Wise Sales](region_wise_sales.png)
- East region generated the highest sales.

### Product-wise Sales
![Product Wise Sales](product_wise_sales.png)
- Mobile is the top revenue-generating product.

### Salesperson-wise Sales
![Salesperson Wise Sales](salesperson_wise_sales.png)
- Sita is the top-performing salesperson.

## Conclusion
This project demonstrates Excel-based data analysis using pivot tables to derive meaningful business insights.
